# ⚠️ IMPORTANT: 3D Model Required

This project requires a 3D room model to function properly.

## You must add a file named `room.glb` to this folder.

### Where to get a 3D room model:

1. **Free 3D Model Sites:**
   - [Sketchfab](https://sketchfab.com/) - Search for "room" with "Downloadable" filter
   - [Poly Pizza](https://poly.pizza/) - Free low-poly 3D models
   - [Free3D](https://free3d.com/) - Free 3D models

2. **Create Your Own:**
   - Download [Blender](https://www.blender.org/) (free)
   - Model a simple room
   - Export as GLB format

3. **Quick Start (Placeholder):**
   - You can start with ANY GLB model temporarily
   - Just rename it to `room.glb`

### Requirements:
- File name: `room.glb`
- Format: GLB (GLTF Binary)
- Recommended size: < 5MB
- Should be positioned at origin (0, 0, 0)

### After adding room.glb:
1. Run `npm install` in the project root
2. Run `npm run dev`
3. Your portfolio should load with the 3D room!
